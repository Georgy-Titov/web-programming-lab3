# Web-программирование. Лабораторная работа №3 — 2048

Учебный проект, разработанный в рамках дисциплины **«Web-программирование»** в Университете ИТМО.

## Описание проекта

Проект представляет собой веб-приложение «2048 — WEB» — популярную головоломку, в которой игрок объединяет плитки с одинаковыми числами для получения плитки со значением 2048.
Приложение реализовано с использованием чистого JavaScript и DOM API без применения сторонних фреймворков.

## Основной функционал

* **Игровой процесс** — перемещение плиток в четырех направлениях (вверх, вниз, влево, вправо) с объединением одинаковых чисел.

* **Система счета** — начисление очков за каждое успешное объединение плиток.

* **Управление** — поддержка клавиатуры (стрелки) для десктопных устройств и сенсорного управления (свайпы) для мобильных устройств.

* **Адаптивные элементы управления** — кнопки направления для десктопов скрываются на мобильных устройствах.

* **Отмена хода** — возможность отменить последний ход с восстановлением предыдущего состояния игры.

* **Сохранение прогресса** — автоматическое сохранение состояния игры в localStorage при каждом ходе.

* **Система рекордов** — таблица лидеров с сохранением топ-10 результатов игроков.

* **Визуальные эффекты** — плавные анимации появления и объединения плиток.

* **Адаптивный дизайн** — интерфейс корректно отображается на разных устройствах и размерах экранов.

* **Модальные окна** — кастомные диалоговые окна для окончания игры, подтверждения действий и просмотра таблицы лидеров.

## Используемые технологии

* **HTML** — семантическая разметка игрового интерфейса и модальных окон.

* **CSS** — оформление игрового поля, плиток, анимации и адаптивный дизайн.

* **JavaScript** — игровая логика, управление состоянием, обработка пользовательского ввода.

* **DOM API** — динамическое обновление игрового поля и взаимодействие с элементами страницы.

* **LocalStorage API** — сохранение состояния игры и таблицы рекордов в браузере.
  
## Архитектура проекта


* `index.html` — минимальный HTML-каркас и подключение скрипта
* `style.css` — оформление интерфейса, анимации и адаптивность
* `script.js` — логика приложения, работа с DOM и LocalStorage
